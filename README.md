# EdgeForge Web

Frontend application for **EdgeForge**, a model-driven sports analytics platform that surfaces structured betting insights using recent player performance data and historical hit rates.

**Live site:**  
https://edgeforge.co

EdgeForge prioritizes **transparency, repeatability, and long-term tracking** over high-volume pick generation.

---

# Features

- Daily **free picks** generated by the analytics engine
- **Historical performance tracking** for player props
- Market-by-market win rate breakdowns
- Player-specific betting analytics
- Secure authentication with **Supabase**
- Subscription model with **Stripe** (in progress)

Production environment variables are configured in Vercel.
